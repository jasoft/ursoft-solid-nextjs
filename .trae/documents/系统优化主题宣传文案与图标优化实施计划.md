## 目标
- 围绕“系统优化”主题（清理注册表、清除垃圾、加速启动、隐私擦除、防护与稳定）构建整站宣传文案与图标体系，提升下载转化。
- 用统一风格的高质量图标替换 Integration 等模块的占位图形，匹配网站的 Windows/Fluent 视觉风格。

## 文案策略
- 语调：面向 C 端用户，直接、可信、结果导向，避免术语堆叠。
- 关键词：快速、彻底、干净、安全、无痕、轻松、Windows 兼容。

## 文案交付（示例，写入 content.ts 后组件引用）
- Hero 主标题/副标题/CTA：
  - 主标题："卸载更彻底，电脑更干净"
  - 副标题："一键清理注册表残留与系统垃圾，让 Windows 更快更稳定"
  - CTA 按钮："立即免费下载"
- FunFact 标题/描述：
  - 标题："全球 Windows 用户一致好评"
  - 描述："安全卸载与深度清理并行，持久保持系统畅快运行"
- Integration 标题/副标题/描述：
  - 标题："全面协同系统工具"
  - 副标题："与 Windows 功能无缝配合"
  - 描述："卸载应用、清理注册表、管理启动项、擦除隐私与垃圾，稳定高效"
- 其他模块（Features/Download/Support）按同风格统一描述与 CTA 词汇。

## 图标集（拟下载并本地化）
- 风格建议：Icons8 Windows/Fluent/Windows 11 Color 系列，统一扁平、明亮、可读性强。
- 图标清单：
  - Windows 徽标（Windows 11）
  - 注册表编辑器（Registry Editor）
  - 垃圾清理（Broom/Trash Clean）
  - 启动项管理（Startup/Power/Boot）
  - 隐私擦除（Shredder/Shield/Privacy）
  - 性能加速（Speed Gauge）
  - 强制卸载/修复卸载（Force Remove/Fix）
- 技术规范：PNG（512px 优先，必要时 256/128），透明背景；命名示例：`registry.png`、`cleanup.png`、`startup.png`、`privacy.png`、`speed.png`、`force-remove.png`、`windows.png`。
- 版权合规：优先选用允许商用/无需署名的资源；如 Icons8 需署名则在页脚或关于页添加 Attribution（可由我统一新增）。

## 实施步骤
1. 选型与下载：从 Icons8 等站点检索并下载上述图标，保存到 `public/images/integration/`。
2. 内容结构：在 `app/content.ts` 增加 `integrationHeader`、`integrationIcons`（替换为新图标路径与 alt），同时完善 Hero/FunFact/CTA 文案变量。
3. 组件改造：
   - `components/Integration/` 引用新的 `integrationHeader` 和 `integrationIcons`，卡片样式统一（圆角、阴影、描边，支持暗色）。
   - 如需在 Features 中复用同一图标体系，也同步更新。
4. 视觉一致性：统一图标尺寸（例如 48–64px），颜色与主色（`--color-primary`）协调；保证暗色模式对比度。
5. 交互与 CTA：按钮采用主色渐变/描边与轻动画，突出“免费下载”。

## 验证
- 视觉：检查图标清晰度、大小、对齐与响应式；深浅色模式一致。
- 性能：图片本地化后加载快速；体积控制在合理范围。
- 合规：核对图标许可与署名要求，必要时增加 Attribution。
